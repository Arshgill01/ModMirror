# Wave 08: Quiz Feedback And Team Norm Scorecard

Status: complete

## Scope

Wave 08 adds deterministic answer feedback and a non-punitive aggregate summary
to the onboarding quiz created in Wave 07.

## Delivered

- Existing calibration answer API integration from the quiz form.
- Feedback for aligned, acceptable alternative, and review-recommended answers.
- Aggregate counts for completed, aligned, acceptable alternative, and review
  recommended outcomes.
- Explicit no-leaderboard UI copy.
