# Wave 11 - Runtime Rehearsal Matrix

## Objective

Run a disciplined Devvit runtime rehearsal for all non-destructive product paths
that are now implemented.

## Deliverables

- Playtest version recorded.
- Runtime matrix updated only for actual observations.
- Runtime notes for:
  - launch card
  - dashboard navigation
  - health score
  - Mirror Scan/demo scan
  - policy agreement
  - Apply Policy preview/log-only receipt
  - override review
  - Team Calibration
  - Settings diagnostics
- Bug list for anything observed.

## Guardrails

- Non-destructive only unless user gives explicit approval.
- No submission screenshots or video work.
- Do not claim native mobile proof unless native app was actually used.

## Acceptance Criteria

- Runtime evidence exists for every non-destructive path tested.
- Failures are reproducible or clearly described.
- Any code fixes from rehearsal pass full validation.
