# Agent G — Final Runtime + Visual Review

## Goal

Perform a ruthless final review of the running app.

## Process

1. Launch app in playtest.
2. Capture screenshots of:
   - inline card,
   - expanded dashboard/Command Center,
   - demo scenario,
   - policy health,
   - override review,
   - case packet,
   - digest page,
   - settings/runtime panel.
3. Compare screenshots against:
   - docs/UX_SPEC.md
   - docs/DESIGN_SYSTEM.md
   - docs/PRODUCTIZATION_ACCEPTANCE_CHECKLIST.md
   - docs/LAUNCH_READINESS_CHECKLIST.md
4. Fix issues.

## Use Tools

If available:

- browser/playwright skill,
- uncodexify skill,
- Gemini CLI design review,
- tmux panes.

## Visual Bar

The UI must not look like:

- a raw dev dashboard,
- a default card grid,
- cramped inside a Reddit post,
- a flat tab dump,
- generic AI SaaS slop.

It should look like:

- a focused moderation command center,
- operational,
- Reddit-native,
- clear,
- dense but readable,
- action-oriented.
